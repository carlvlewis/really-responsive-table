Really Responsive Table

The table will become scrollable and automatically lock columns marked with data-auto-lock or data-auto-lock-rear as soon as the table becomes wider than its container. Columns are lockable/unlockable using controls that appear above. At a configurable breakpoint (767px by default), the table will become stacked.

Bootstrap is used here for quick table styles, but the table's functionality doesn't depend on it. Tested Chrome, Firefox, IE10+, Android, Desktop Safari. Works in Mobile Safari when not inside of CodePen.
